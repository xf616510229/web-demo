#前端实例：京东

##项目规划：

###项目文件夹：
1. 首页 index.html
2. 样式 css文件夹 css文件 全局样式(base.css/global.css)
3. 图片 images文件夹
4. 特效 js文件夹

##开始项目
1. 将项目基本样式引入到index.html中
```html
    <!--引入css  注意样式表的顺序-->
    <link rel="stylesheet" href="css/base.css">
    <link rel="stylesheet" href="css/index.css">
```
2. 设置图标
```html
    <!--设置图标-->
    <link rel="shortcut icon" href="favicon.ico">
```
